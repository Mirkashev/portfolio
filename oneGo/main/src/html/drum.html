наш stack:
swift надёжный и интуитивно понятный язык программирования от Apple, при помощи которого можно создавать приложения для всех iOS устройств.
flutter кроссплатформенное решение, благодаря которому можно написать один код на две системы. Позволяет частично уменьшить затраты времени на проектирование бизнес-логики и тестирование.
Kotlin представляет современный, статически типизированный и один из самых быстроразвивающихся языков программирования. Его используют для создания мобильных приложений на Android, язык при помощи которого можно создавать приложение на всех Android устройствах.
flutter кроссплатформенное решение, благодаря которому можно написать один код на две системы. Позволяет частично уменьшить затраты времени на проектирование бизнес-логики и тестирование.
Node.js Node или Node.js — программная платформа, основанная на движке V8, превращающая JavaScript из узкоспециализированного языка в язык общего назначения.
GO Go — компилируемый многопоточный язык программирования, разработанный внутри компании Google.
